# Devoir GTI650

### Ce projet à été créé avec Python 3.13.7 et les dépendances installés avec pip 25.2

### Ce projet est accessible sur un repo GitHub public accessible au lien suivant:

https://github.com/taste3/GTI650-devoir-equipe3

### Install requirements: run ```pip install -r requirements.txt```

### Après avoir installé les requis, vous pouvez lancer le sudoku2x2 avec la commande:

````python sudoku2x2.py```

### Ou le sudoku3x3 avec la commande:

````python sudoku3x3.py```
